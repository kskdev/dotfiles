# dotfiles

いろんな設定設定ファイルを公開できる範囲で公開．
プライベート用に使っていたものをどこでも使えるように公開．
あくまでも個人用．意味不明な変更も多数入る可能性が高い．


目的
- サーバやGCPなどで動かすときの初期設定の簡略化．`git clone` で簡単にDLしたい．
