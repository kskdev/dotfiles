# About

設定を端末間で同期するための `Sync` などが使えない環境で，最低限の設定を記述

- VSCode 用の設定と VSCode のフォークプロジェクトである Antigravity の設定に対応

# How to use

`install_extensions.bat` を実行をするだけ

- 拡張機能群をインストール(インストールリスト: `./extensions.txt`)
- 設定ファイルを定義
  - `Settings.json`
  - `keybindings.json`
