- このレポジトリは各種開発環境やアプリの設定ファイルを扱う
  - レポジトリの管理者が様々な環境で再現性のあるアプリ設定を補助することを目的としている
  - 様々な環境: macOS, Ubuntu, windows などの実行環境の変動や，git clone しか許可されていない環境での利用を想定
  - ただし，可能な限りでの対応とする
- 機密情報はレポジトリの管理対象外であり，それらの情報が含まれているファイルはコミットを禁止する
- 管理コストを増やさないために，ユーザが既知であるツール以外の新しいツールやアプリの導入は非推奨とする
- ユーザの備忘録を兼ねているため，記載されているコメントなどはなるべくそのまま残す
- apt 経由のツールなどの設定は，安定版のツールで動作することを前提とする
  - vim のようにビルド前提のものは，その限りではない
  - 簡易なインストーラを作成する場合は，初見で分かるように作成する
- 逐次 README.md を更新すること
  - レポジトリルートは完結な内容 + 大きな変更があった時のログを更新
  - 各ディレクトリごとは，使い方の README を更新(修正のログは不要)
- コメントは基本的に日本語で記述する
